

### TypingTutor 

#### 概要

このプロジェクトは、Streamlit を使用して作成されたシンプルなタイピング練習・測定アプリケーションです。ユーザーは表示されたことわざや格言をタイピングし、その速度（WPM）と正確性を測定できます。

#### 機能

  * **ランダムな練習テキスト:** 毎回異なることわざが表示されます。
  * **リアルタイムな測定:** タイピング中にリアルタイムの WPM（Words Per Minute）が表示されます。
  * **完了後の結果表示:** タイピング完了後、最終的な WPM と正確性（%）が計算されて表示されます。
  * **シンプルな UI:** 直感的で使いやすいインターフェースを提供します。

#### セットアップ方法

##### 1\. 依存関係のインストール

このアプリケーションは、`streamlit` のみが必要です。以下のコマンドでインストールできます。

```bash
pip install streamlit
```

##### 2\. アプリケーションの実行

上記で提供された Python コードを `app.py` という名前で保存し、以下のコマンドを実行します。

```bash
streamlit run app.py
```

コマンドを実行すると、ローカルブラウザでアプリケーションが起動します。

#### 使い方

1.  アプリケーションを起動すると、画面上部にことわざが表示されます。
2.  「ここにタイピング」と書かれた入力ボックスに、表示されたテキストを正確に入力し始めます。
3.  タイピングを始めると、サイドバーにリアルタイムの WPM と経過時間が表示されます。
4.  すべてのテキストを正確に入力し終えると、タイピングが完了し、最終的な結果が表示されます。
5.  「再挑戦」ボタンをクリックすると、新しいテキストで再度タイピングを始めることができます。

#### 備考

  * **WPM（Words Per Minute）** は、「入力した単語数 / 経過時間（分）」で計算されます。
  * **正確性（Accuracy）** は、「正しく入力した文字数 / ユーザーが入力した文字数」で計算されます。

このコードは、Streamlit のセッションステートを効果的に活用し、タイピングの開始・終了状態やユーザー入力を管理しています。さらに機能を拡張したい場合は、より多くの練習テキストを追加したり、履歴を保存する機能などを実装したりすることが可能です。
