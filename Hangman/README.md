
# Hangman 

このプロジェクトは、PythonのStreamlitフレームワークを使用して構築されたシンプルなハングマンゲームです。ウェブブラウザ上で手軽に遊ぶことができます。

## アプリケーションの機能

- **ランダムな単語の選択**: 毎回異なる単語が出題されます。
- **インタラクティブなUI**: Streamlitのウィジェットを使って、ユーザーが文字を推測できます。
- **視覚的なフィードバック**: 間違えるたびにハングマンのイラストが描画され、ゲームの進行状況が一目でわかります。
- **勝利・敗北判定**: 正解すると風船が舞い、間違えすぎるとゲームオーバーになります。
- **リプレイ機能**: ゲーム終了後、「もう一度プレイする」ボタンで簡単に新しいゲームを開始できます。

## セットアップと実行方法

### 1. 必要な環境

- Python 3.7以上がインストールされていること。

### 2. 依存ライブラリのインストール

このアプリケーションを実行するには、`streamlit` ライブラリが必要です。以下のコマンドでインストールしてください。

```bash
pip install streamlit
````

### 3\. アプリケーションの実行

以下のコマンドをターミナルで実行してください。

```bash
streamlit run app.py
```

コマンド実行後、ローカルサーバーが起動し、自動的にブラウザでアプリケーションが開きます。

## ゲームの遊び方

1.  アプリケーションを開くと、単語が `_` で表示されます。
2.  「アルファベットを推測してください」という入力ボックスに、推測するアルファベット一文字を入力し、**Enterキー**を押します。
3.  正解すると、単語の `_` がその文字に置き換わります。
4.  不正解の場合、ハングマンのイラストにパーツが一つ追加されます。
5.  全ての文字を当てるか、ハングマンが完成するまでゲームを続けます。

楽しんでください！
